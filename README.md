## PlayMusic 🎶
O _PlayMusic_ é uma plataforma voltada para o gerenciamento de músicas, ideal para quem gosta de organizar suas faixas favoritas. O foco do aplicativo não é reproduzir músicas, mas oferecer uma maneira prática e intuitiva de criar, gerenciar e personalizar playlists, facilitando o acesso às suas canções preferidas. Vale lembrar que o _PlayMusic_ ainda está em processo de ajustes.
